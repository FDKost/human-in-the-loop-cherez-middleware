# Human-in-the-Loop через middleware

## Requirements
- [high] Внедрить LangChain‑агентскую архитектуру с HumanInTheLoopMiddleware: Добавить в проект модуль LangChain, настроить агента и подключить HumanInTheLoopMiddleware для обработки пользовательских запросов в режиме реального времени.
- [high] Реализовать MemorySaver для сохранения пауз: Создать компонент MemorySaver, который будет сохранять состояние агента при паузах и восстанавливать его при возобновлении работы.
- [high] Добавить команду Command для возобновления работы: Разработать команду Command, позволяющую пользователю возобновить работу агента после паузы, используя сохранённые данные.
- [high] Обновить requirements.txt, добавить langgraph: Внести в файл requirements.txt зависимость langgraph, чтобы обеспечить корректную работу всех компонентов проекта.
